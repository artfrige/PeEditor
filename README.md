## PeEditor Introduction
PeEditor is simple HTML5 Rich Text Editor

## Initialization
new PeEditor("#YOUR_CONTAINER_ID")

## Methods
[INSTANCE_NAME].returnEditValue()  
Returns editor's innerHTML code
